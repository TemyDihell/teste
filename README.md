
# Dashboard Comercial - Streamlit Cloud

## Funcionalidades

- Dashboard de vendas
- Ranking de vendedores
- Desempenho por equipe
- Desempenho por cidade
- Positivação de clientes
- Batalha naval por fabricante

## Estrutura da Planilha

Campos obrigatórios:

- Data
- Ano
- Mês
- Vendedor
- Equipe
- Cidade
- Cliente
- Fabricante
- Produto
- Quantidade
- Valor Venda

## Como subir no Streamlit Cloud

1. Crie conta no GitHub
2. Crie um repositório
3. Envie todos os arquivos
4. Entre no Streamlit Cloud
5. Clique em "New App"
6. Selecione o repositório
7. Escolha:
    - Branch: main
    - Main file path: app.py
8. Clique em Deploy

O sistema ficará online automaticamente.
